kitty & mouse

kitty & mouse - an exciting multi-level puzzle

game with which you can relax and spend a cozy evening.

the goal of the game is that we, playing as a kitten,

catch the mouse in a certain number of moves

on the field 8 by 8. but not everything is so simple,

with each level the game becomes more complicated.

happy hunting kittens!
